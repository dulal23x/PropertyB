# FEATURE_INQUIRIES

## Purpose
Capture buyer/visitor interest and route it to the business/admin team.

## Public Inquiry Form
Fields:

- name
- phone
- email optional
- message optional
- preferred contact method

## Rules
- Inquiry can be submitted only for approved listings.
- Inquiry stores visitor contact data for admins.
- Listing owner does not receive visitor details directly in MVP.
- Admin receives internal email notification.
- Visitor receives confirmation email only when email is supplied.

## Admin Handling
Admin can mark inquiries:

- new
- contacted
- closed
- spam
