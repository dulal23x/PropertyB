# FEATURE_ADMIN_DASHBOARD

## Purpose
Admin dashboard is the operations center for listing approvals, inquiries, users, and email operations.

## Routes
- `/admin`
- `/admin/properties`
- `/admin/properties/[id]`
- `/admin/inquiries`
- `/admin/users`
- `/admin/email-hub`
- `/admin/email-hub/compose`
- `/admin/email-hub/templates`
- `/admin/settings`

## Dashboard Metrics
- pending listings
- approved listings
- rejected listings
- unpublished listings
- new inquiries
- total users
- emails sent today

## Approval Checklist
- title is clear
- description is useful
- location is meaningful
- price or call-for-price is valid
- at least one image exists
- public phone is configured
- owner phone is not public
- no spam/prohibited content

## Required Admin Actions
- approve
- reject with note
- unpublish
- archive
- edit basic listing fields
- mark inquiry contacted/closed/spam
- manage users
- view email logs/templates
