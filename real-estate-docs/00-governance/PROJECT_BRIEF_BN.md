# Project Brief - RealEstateSite MVP

## লক্ষ্য
`C:\realestatesite`-এ একটি real estate listing MVP তৈরি করা হবে যেখানে ব্যবহারকারী property/land listing submit করবে, admin review করে approve করলে listing public search-এ দেখা যাবে।

## MVP Scope
- User signup/login
- Client listing dashboard
- Public property search
- Public property detail page
- Business phone CTA
- Inquiry form
- Admin listing approval dashboard
- Admin inquiry management
- Email logs/templates/notifications

## Non-MVP
- Payment
- MLS
- Agent marketplace
- Live chat
- Mobile app
- Ownership verification automation
- Resume/job/application flows

## Critical Product Rule
Public listing page-এ listing owner/client phone number দেখানো যাবে না। Public phone হবে business real estate phone.
