# Definition Of Done

A change is done only when:

- It is implemented in `C:\realestatesite`, not the reference repo.
- Relevant docs and contracts are updated.
- Backend tests or route smoke checks pass.
- Frontend lint/build or targeted browser smoke passes.
- Public API privacy rules are verified.
- Admin-only actions reject non-admin users.
- No secrets are committed.
- Changelog and decisions are updated for meaningful behavior changes.

MVP is done only when:

- User signup/login works.
- Client can create and submit listings.
- Admin can approve/reject/unpublish listings.
- Approved listings are publicly searchable.
- Public listing shows business phone only.
- Inquiry flow reaches admin.
- Email logs show core notifications.
- Dashboard and admin routes work on desktop and mobile.
